[ 플러터 원본소스를 웹배포용 소스로 빌드하기]

1. 빌드할 플러트 프로젝트 위치로 이동한다! (cd...)

2. 플러터 명령어로 웹빌드한다.
플러터 빌드 명령어 : 특정플랫폼에 맞게 빌드
웹은 web
- flutter build web

3. 빌드후 작업하기 
(1) build 폴더 하위 web폴더 아래 index.html파일을 연다
    <base href='/'>
    소스 중 이부분이 루트를 연결하는 부분이다.
    만약 서버에 올리면 서버 루트 주소를 넣고
    라이브 서버로 연습중이면 그 주소를 띄워보고
    복사하여 넣어준다.

->>> 아래 이미지 경로 맞추기는 플러트에서 상위폴더부터 경로를 
쓰지 않은 경우 웹 경로가 다르게 표시되는 사례이다!
->>> 따라서 웹 경로를 상위부터 써주면 (2)을 할 필요가 없다.

(2) 대부분이 출력되지만 이미지가 출력되지 않는 경우
개발자 모드에서 이미지 404에러 경로를 보고
실제 경로가 맞지 않는 폴더를 이동하여 이미지 경로를 맞춰준다
(여기서는 assets/assets/images -> assets/images)
